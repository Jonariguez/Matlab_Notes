# MATLAB数组
## 按照元素个数和排列方式分类
1. 没有元素的空数组
2. 只有一个元素的标量，实际上是一行一列的数组
3. 只有一行或者一列的向量，分别叫做行向量或者列向量，统称为一维数组
4. 普通的二维数组
5. 多维数组

**size()函数**
* 如果按照只有一个返回值的形式调用，如sz=size(A)，将会返回一个n维向量，其中每一个数表示A的每一维的大小
* 如果按照只有多个返回值的形式调用，如[row,col]=size(A)，每一个数表示A的每一维的大小。为避免出错，给的个数应当和A的维数相同。<br>

**length()函数**
length()返回数组的最大维值。

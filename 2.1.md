# 简单计算

* 直接输入法
  ```
  >>0.5\*pi\*5^2\*10
  ans=
     392.6991
  ```
* 存储变量法

  ```
  >>s=0.5\*pi\*5^2
  s=
    39.2699
  >>v=s*10
  v=
    392.6991
  ```

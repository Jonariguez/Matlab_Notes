# 常用的操作命令和快捷键
### 常见操作命令
| 命令 | 功能 |
| :---: | :---: |
| cd | 显示或改变工作目录 |
| clc | 清空命令窗口 |
| clear | 清除工作区的变量 |
| clf | 清除图像窗口 |
| diary | 日志文件命令 |
| dir | 显示当前目录下的文件 |
| disp | 显示变量或者文字的内容 |
| echo | 命令窗口信息显示开关 |
| hold | 图形保持命令 |
| load | 加载指定文件中的变量 |
| pack | 整理内存碎片 |
| path | 显示搜索目录 |
| quit | 退出MATLAB |
| save | 保持内存变量或者工作区变量 |
| type | 显示文件内容 |
